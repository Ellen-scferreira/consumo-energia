## 📖 Sobre o projeto

Este projeto em **Python** calcula o consumo mensal de energia elétrica de um aparelho doméstico.

O usuário informa:

- nome do aparelho
- potência em watts
- tempo médio de uso diário

O programa calcula o consumo mensal em **kWh**.

## 📐 Fórmula utilizada
consumoMensal = (potencia * horasDia * 30) / 1000
## ▶️ Como executar

Abra o terminal e execute:
python app.py

## 🎯 Exemplo

Aparelho: Geladeira  
Consumo estimado: 45 kWh/mês
